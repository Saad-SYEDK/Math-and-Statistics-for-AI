# Math-and-Statistics-for-AI

This repository provides concise notes on the essential mathematics and statistics required for Artificial Intelligence, Machine Learning, and Data Science. Designed for quick review and practical understanding, this collection covers the foundational pillars that drive modern AI models. 

## Topics to be Covered

- Basics: Descriptive vs inferential statistics, continuous vs discrete data,
- nominal vs ordinal data
- Linear Algebra: Vectors, Metrices, Eigenvalues and Eigenvectors
- Calculus: Basics of integral and differential calculus
- Basic plots: Histograms, pie charts, bar charts, scatter plot etc.
- Measures of central tendency: mean, median, mode
- Measures of dispersion: variance, standard deviation
- Basics of Probability
- Distributions: Normal distribution
- Correlation and covariance
- Central limit theorem
- Hypothesis testing: p value, confidence interval, type 1 vs type 2 error, Z test
